1. パラメータの書き方

コマンドライン、Java アプレットの <param>、Java Web Start の <argument> および設定ファイルで共通のキーワードを用いてパラメータを設定します。

1.1. コマンドライン

-キーワード=設定値

1.2. Java アプレット

<param name="キーワード" value="設定値">

1.3. Java Web Start

<argument>-キーワード=設定値</argument>

1.4. 設定ファイル

キーワード=設定値

2. パラメータの説明

2.1. boot

起動デバイスを指定します。ファイル名またはディレクトリ名を指定すると種類に応じて該当するデバイスの空いているユニットに割り当てられます。

設定値 [初期値]

std | fdN | hdN | scN | hfN | rom$～ | ram$～ | ファイル名 | ディレクトリ名

  • std … 標準 (FDD → 内蔵 SASI または 内蔵 SCSI → 拡張 SCSI を含む ROM → SRAM)
  • fdN … フロッピーディスク
  • hdN … SASI ハードディスク
  • scN … SCSI ハードディスク / CD-ROM
  • hfN … ホストファイルシステム
  • rom$～ … ROM 起動アドレス
  • ram$～ … RAM 起動アドレス
  • ファイル名 … フロッピーディスク、SASI ハードディスク、SCSI ハードディスク / CD-ROM のいずれかに割り当てるイメージファイル
  • ディレクトリ名 … ホストファイルシステムに割り当てるディレクトリ

2.2. cgrom

CGROM ($00F00000～$00FBFFFF、768KB) のイメージファイルを指定します。

設定値 [初期値]

ファイル名 [CGROM_XEiJ.DAT,CGROM.DAT]

2.3. clock

MPU の動作周波数 (MHz) を指定します。model を書いたときはその動作周波数になります。指定がなければ model に従います。

設定値 [初期値]

1～1000 | EXPERT | SUPER | XVI | Compact | Hybrid | X68030 | 060turbo

  • 1～1000 … 1MHz～1000MHz
  • EXPERT … 10MHz
  • SUPER … 10MHz
  • XVI … 16.7MHz
  • Compact … 25MHz
  • Hybrid … 33.3MHz
  • X68030 … 25MHz
  • 060turbo … 50MHz

2.4. config

設定をデフォルトに戻すときに指定します。

設定値 [初期値]

[""] | default

  • default … 設定をデフォルトに戻す

2.5. cutfc2pin

FC2 ピンをカットするかどうか指定します。FC2 ピンをカットすると物理アドレス空間全体がユーザモードでアクセスできるようになります。NetBSD/x68k で X Window System を動かすときに使います。

設定値 [初期値]

on | [off]

  • on … FC2 ピンをカットする
  • off … FC2 ピンをカットしない

2.6. edgeaccel

縁部加速 するかどうか指定します。

設定値 [初期値]

on | [off]

  • on … 縁部加速する
  • off … 縁部加速しない

2.7. fdN (N=0～3)

フロッピーディスクのイメージファイルを指定します。ファイル名の末尾に :R を付けると書き込み禁止になります。

設定値 [初期値]

ファイル名

2.8. fefunc

Human68k が動作しているとき FE ファンクションコールを FE ファンクション命令 として処理するかどうか指定します。

設定値 [初期値]

[on] | off

  • on … FE ファンクションコールを命令として処理する
  • off … FE ファンクションコールを命令として処理しない

2.9. fitinwindow

X68000 の画面の大きさをウインドウに合わせて拡大縮小するかどうか指定します。

設定値 [初期値]

[on] | off

  • on … X68000 の画面の大きさをウインドウに合わせて拡大縮小する
  • off … X68000 の画面の大きさをウインドウに合わせない

2.10. fixedscale

X68000 の画面を表示する大きさを倍率 (%) で指定します。

設定値 [初期値]

10～[100]～1000

  • 10 … X68000 の画面を 1/10 に縮小して表示する
  • 100 … X68000 の画面を等倍で表示する
  • 1000 … X68000 の画面を 10 倍に拡大して表示する

2.11. fpumode

FPU の動作モードを指定します。三倍精度浮動小数点数 を選択できます。

設定値 [初期値]

0 | [1] | 2

  • 0 … FPU なし
  • 1 … 拡張精度浮動小数点数
  • 2 … 三倍精度浮動小数点数

2.12. fullspecfpu

FPU および FPCP を MC68882 と MC68060 のすべての浮動小数点命令を実行できるフルスペック版にするかどうか指定します。

設定値 [初期値]

on | [off]

  • on … FPU および FPCP をフルスペック版にする
  • off … FPU および FPCP を通常版にする

2.13. fullscreen

全画面表示にするかどうか指定します。

設定値 [初期値]

on | [off]

  • on … 全画面表示にする
  • off … 全画面表示にしない

2.14. hdN (N=0～15)

SASI ハードディスクのイメージファイルを指定します。ファイル名の末尾に :R を付けると書き込み禁止になります。

設定値 [初期値]

ファイル名

2.15. hfN (N=0～15)

ホストファイルシステム に割り当てるディレクトリを指定します。ディレクトリ名の末尾に :R を付けると書き込み禁止になります。

設定値 [初期値]

ディレクトリ名

2.16. highmemory

X68030 のハイメモリ $01000000-$01FFFFFF (16MB) の容量を指定します。九十九電気 (ツクモ) の TS-6BE16 に相当します。

設定値 [初期値]

[0] | 16

  • 0 … ハイメモリなし
  • 16 … ハイメモリ $01000000-$01FFFFFF (16MB)

2.17. highmemorydata

X68030 のハイメモリの内容を gzip+base64 で変換した文字列です。 highmemorysave が on のとき自動的に保存されます。

設定値 [初期値]

[""]

2.18. highmemorysave

X68030 のハイメモリの内容を設定ファイルに保存するかどうか指定します。

設定値 [初期値]

on | [off]

  • on … X68030 のハイメモリの内容を設定ファイルに保存する
  • off … X68030 のハイメモリの内容を設定ファイルに保存しない

2.19. hhssbb

ウインドウやキーボードの色を指定します。

設定値 [初期値]

Hmin,Hmax,Smin,Smax,Bmin,Bmax [667,667,700,300,0,1000]

  • Hmin … 色相 (hue) の下限 (0～1000)
  • Hmax … 色相 (hue) の上限 (0～1000)
  • Smin … 彩度 (saturation) の下限 (0～1000)
  • Smax … 彩度 (saturation) の上限 (0～1000)
  • Bmin … 明度 (brightness) の下限 (0～1000)
  • Bmax … 明度 (brightness) の上限 (0～1000)

2.20. hostspixelunits

エクスクルーシブマウスモード のときマウスカーソルの移動速度をホストマシンの画素単位に合わせるかどうか指定します。ホストマシンの画素単位に合わせると、マウスカーソルの移動速度が X68000 の画面の表示倍率の影響を受けにくくなります。

設定値 [初期値]

on | [off]

  • on … ホストマシンの画素単位に合わせる
  • off … X68000 の画素単位に合わせる

2.21. intermittent

間欠描画の間隔を指定します。描画の頻度を下げると表示がぎこちなくなりますが動作は軽くなります。

設定値 [初期値]

[0] | 1 | 2 | 3 | 4

  • 0 … 変化したフレームをすべて描画する
  • 1 … 変化したフレームを 1/2 描画する
  • 2 … 変化したフレームを 1/3 描画する
  • 3 … 変化したフレームを 1/4 描画する
  • 4 … 変化したフレームを 1/5 描画する

2.22. interpolation

画面の補間アルゴリズムを指定します。

設定値 [初期値]

nearest | [bilinear] | bicubic

  • nearest … 最近傍補間
  • bilinear … 線形補間
  • bicubic … 三次補間

2.23. iplrom

IPLROM ($00FE0000～$00FFFFFF、128KB) のイメージファイルを指定します。 model で指定した機種に応じて IPLROM.DAT、IPLROMXV.DAT、IPLROMCO.DAT、IPLROM30.DAT のいずれかがリソースから読み込まれるので、通常は指定する必要はありません。

設定値 [初期値]

ファイル名

2.24. joyauto

ジョイスティックポートが継続的にアクセスされている間だけ割り当てられたキーの操作をジョイスティックの操作と見なすかどうか指定します。ジョイスティックポートが継続的にアクセスされているかどうかで割り当てられたキーの操作をジョイスティックの操作とキーボードの操作に自動的に振り分けることができます。

設定値 [初期値]

[on] | off

  • on … ジョイスティックポートが継続的にアクセスされている間だけ割り当てられたキーの操作をジョイスティックの操作と見なす
  • off … 割り当てられたキーの操作を常にジョイスティックの操作と見なす

2.25. joyblock

ジョイスティックの操作と見なされたキーの操作をキーボードの入力から取り除くかどうか指定します。

設定値 [初期値]

[on] | off

  • on … ジョイスティックの操作と見なされたキーの操作をキーボードの入力から取り除く
  • off … ジョイスティックの操作と見なされたキーの操作をキーボードの入力から取り除かない

2.26. joykey

キーにジョイスティックのボタンを割り当てるかどうか指定します。

設定値 [初期値]

[on] | off

  • on … キーにジョイスティックのボタンを割り当てる
  • off … キーにジョイスティックのボタンを割り当てない

2.27. joymap

ジョイスティックのボタンを割り当てるキーと連射の設定を行います。

設定値 [初期値]

key-code:repeat:delay:interval,… [38,40,37,39,,90,88,,83,68]

  • key-code … キーコード
  • repeat … 連射する (1)
  • delay … 連射開始 (ms)
  • interval … 連射間隔 (ms)

ボタンは上、下、左、右、C、A、B、D、SELECT、RUN の 10 個。ジョイスティック 1 と 2 を合わせて 20 個。

2.28. keyboard

キーボードの種類を指定します。

設定値 [初期値]

none | [standard] | compact

  • none … キーボードなし
  • standard … 標準キーボード
  • compact … コンパクトキーボード

2.29. keydly

キーボードのリピート開始時間を指定します。

設定値 [初期値]

[-1] | 0～15

  • -1 … 既定
  • 0～15 … 200+100*n (ms)

2.30. keyrep

キーボードのリピート間隔を指定します。

設定値 [初期値]

[-1] | 0～15

  • -1 … 既定
  • 0～15 … 30+5*n*n (ms)

2.31. lang

表示言語を指定します。指定がなければ実行環境が日本語のときだけ日本語になります。

設定値 [初期値]

en | ja

  • en … 英語
  • ja … 日本語

2.32. localmemory

060turbo のローカルメモリ $10000000-$1FFFFFFF (16MB～256MB) の容量を指定します。

設定値 [初期値]

0 | 16 | 32 | 64 | [128] | 256

  • 0 … ローカルメモリなし
  • 16 … ローカルメモリ $10000000-$10FFFFFF (16MB)
  • 32 … ローカルメモリ $10000000-$11FFFFFF (32MB)
  • 64 … ローカルメモリ $10000000-$13FFFFFF (64MB)
  • 128 … ローカルメモリ $10000000-$17FFFFFF (128MB)
  • 256 … ローカルメモリ $10000000-$1FFFFFFF (256MB)

2.33. localmemorydata

060turbo のローカルメモリの内容を gzip+base64 で変換した文字列です。 localmemorysave が on のとき自動的に保存されます。

設定値 [初期値]

[""]

060turbo のローカルメモリの内容を設定ファイルに保存するかどうか指定します。

設定値 [初期値]

on | [off]

  • on … 060turbo のローカルメモリの内容を設定ファイルに保存する
  • off … 060turbo のローカルメモリの内容を設定ファイルに保存しない

2.35. memory

設定値 [初期値]

1 | 2 | 4 | 6 | 8 | 10 | [12]

  • 1 … メインメモリ $00000000-$000FFFFF (1MB)
  • 2 … メインメモリ $00000000-$001FFFFF (2MB)
  • 4 … メインメモリ $00000000-$003FFFFF (4MB)
  • 6 … メインメモリ $00000000-$005FFFFF (6MB)
  • 8 … メインメモリ $00000000-$007FFFFF (8MB)

  • 12 … メインメモリ $00000000-$00BFFFFF (12MB)

2.38. mhz

MPU の任意の周波数 (MHz) のスピナーの初期値を指定します。model にある動作周波数で起動したときスピナーに設定されます。

設定値 [初期値]

1～[100]～1000

  • 1 … 1MHz
  • 100 … 100MHz
  • 1000 … 1000MHz

2.39. model

機種を指定します。デフォルトの X68000 Hybrid は X68000 に X68030 の ROM を載せた架空の機種です。機種に応じて IPLROM のバージョン、内蔵ハードディスクの種類、MPU の種類および動作周波数が設定されます。IPLROM*.DAT ファイルはリソースから読み込まれるので別途用意する必要はありません。060turbo のときも 1.3 にパッチを当てて起動するので実機の ROM は不要です。

設定値 [初期値]

EXPERT | SUPER | XVI | Compact | [Hybrid] | X68030 | 060turbo

  • EXPERT … X68000 EXPERT。IPLROM は 1.0、内蔵ハードディスクは SASI、MPU は MC68000、動作周波数は 10MHz
  • SUPER … X68000 SUPER。IPLROM は 1.0、内蔵ハードディスクは SCSI、MPU は MC68000、動作周波数は 10MHz
  • XVI … X68000 XVI。IPLROM は 1.1、内蔵ハードディスクは SCSI、MPU は MC68000、動作周波数は 16.7MHz
  • Compact … X68000 XVI Compact。IPLROM は 1.2、内蔵ハードディスクは SCSI、MPU は MC68000、動作周波数は 25MHz
  • Hybrid … X68000 Hybrid。IPLROM は 1.3、内蔵ハードディスクは SCSI、MPU は MC68000、動作周波数は 33.3MHz
  • X68030 … X68030。IPLROM は 1.3、内蔵ハードディスクは SCSI、MPU は MC68EC030、動作周波数は 25MHz
  • 060turbo … X68030+060turbo。IPLROM は 1.6、内蔵ハードディスクは SCSI、MPU は MC68060、動作周波数は 50MHz

  • off … FLOATn.X を組み込む

設定値 [初期値]

ファイル名

2.49. rom30

X68030 の ROM の拡張部分 ($00FC0000～$00FDFFFF、128KB) のイメージファイルを指定します。ROM30.DAT など。

設定値 [初期値]

ファイル名

2.50. romdb

on | [off]

2.51. saveonexit

  • off … 終了時に設定を保存しない

2.52. scN (N=0～15)

SCSI ハードディスク / CD-ROM のイメージファイルを指定します。ファイル名の末尾に :R を付けると書き込み禁止になります。

設定値 [初期値]

ファイル名

2.53. scsiex

拡張 SCSI ボードを取り付けるかどうか指定します。

設定値 [初期値]

on | [off]

  • on … 拡張 SCSI ボードを取り付ける
  • off … 拡張 SCSI ボードを取り付けない

設定値 [初期値]

ファイル名

2.55. scsiinrom

内蔵 SCSI ROM ($00FC0000～$00FC1FFF、8KB) のイメージファイルを指定します。SCSIINROM.DAT など。指定がなければリソースから読み込んだ IPLROM30.DAT を使って自動生成するので、通常は指定する必要はありません。

設定値 [初期値]

ファイル名

2.56. seamless

シームレスマウスモード にするかどうか指定します。

設定値 [初期値]

[on] | off

  • on … シームレスマウスモード
  • off … エクスクルーシブマウスモード

2.57. sound

音声を出力するかどうか指定します。

設定値 [初期値]

[on] | off

  • on … 音声を出力する
  • off … 音声を出力しない

2.58. soundinterpolation

音声出力の補間アルゴリズムを指定します。

設定値 [初期値]

thinning | [linear] | constant-area | linear-area

  • thinning … 間引き
  • linear … 線形補間
  • constant-area … 区分定数面積補間
  • linear-area … 線形面積補間

2.59. sram

SRAM ($00ED0000～$00ED3FFF、16KB) のイメージファイルを指定します。SRAM.DAT など。このファイルは終了時に更新されません。SRAM の内容は終了時に設定ファイルに保存されます。SRAM のイメージファイルの読み書きはメニューで行います。

設定値 [初期値]

ファイル名

2.60. sramdata

SRAM の内容を gzip+base64 で変換した文字列です。終了時に自動的に保存されます。

設定値 [初期値]

[""]

設定値 [初期値]

on | [off]

  • off … 負荷率固定モードにしない

2.62. verbose

  • off … 冗長表示しない

2.63. volume

0～[20]～40

  • 0 … 最小
  • 20 … 標準
  • 40 … 最大

2.64. *open

ウインドウが開いているかどうか。* はウインドウの種類 (下記)。終了時に自動的に保存されます。

設定値 [初期値]

on | [off]

  • on … ウインドウが開いている
  • off … ウインドウが閉じている

2.65. *rect

ウインドウの位置と大きさ。* はウインドウの種類 (下記)。終了時に自動的に保存されます。

設定値 [初期値]

x,y,width,height [0,0,0,0]

  • x … x 座標
  • y … y 座標
  • width … 幅
  • height … 高さ

2.66. *stat

iconified | maximized | h-maximized | v-maximized | [normal]

  • maximized … 最大化

  • normal … 普通
  • v-maximized … 垂直方向に最大化
